# Course Square Task1

## Getting Started

### Environment Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/wasuza12345/courseSquare_task1.git
   cd course-square
   ```

2. Start the Docker containers:
   ```bash
   docker-compose up -d
   ```

### Starting the Application
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
